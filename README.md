# Thapar Chat 💬

Chat App For Thapar students

## Features ✨

- Real-time messaging
- Simple and intuitive interface
- Built with Socket.IO and Node.js

## Getting Started 🚀

To run the Thapar Chat app, follow these steps:

1. Clone the repository:
```
git clone https://github.com/your-username/thapar-chat.git
```
2. Navigate to the project directory:
```
cd thapar-chat
```
3. Install the dependencies:
```
npm install
```
4. Start the server:
```
node index.js
```
5. Open your web browser and visit `http://localhost:5000`.

## Usage 📖

1. Enter your name in the "Name" input field.
2. Type your message in the "Message" input field.
3. Click the "Send" button to send your message.

The messages will be displayed in the chat area.
